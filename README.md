# 🌟 Project Overview

The **AI-Powered Education ERP** is a comprehensive enterprise application built using **Java, Spring Boot, Spring Security, Hibernate, Thymeleaf, and MySQL**.

The platform centralizes student, trainer, finance, and administrative operations into a secure role-based system while providing analytics dashboards, payment integration, document management, and intelligent insights.

The project follows a layered architecture with emphasis on security, maintainability, scalability, and clean code practices.

---

# 🏗 Enterprise Architecture

```text
                         Users
                            │
 ┌──────────────┬────────────┼─────────────┐
 │              │            │             │
Admin       Trainer      Student     Accountant
 │              │            │             │
 └──────────────┴────────────┼─────────────┘
                             │
                 Spring Security
          Authentication & Authorization
                             │
                  Spring Boot MVC
                             │
                 Business Service Layer
                             │
              Spring Data JPA / Hibernate
                             │
                        MySQL Database
                             │
              Analytics & AI Integration
```
